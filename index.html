Desenvolva sites e aplicações poderosas de forma simples.


Painel Intuitivo

Feito Para Programadores

Alguns recursos do WPanel CMS


Estrutura inicial

A idéia inicial sobre o WPanel é prover uma estrutura básica para iniciar um projeto de site ou aplicação. Desta forma, ao fazer uma instalação do WPanel CMS você já conta com um site de exemplo com as funcionalidades de um site básico.


Controle do conteúdo

O painel de controle fornece ferramentas para gerenciar todo o conteúdo de um site básico, seja ele um blog ou site institucional com controle de páginas, artigos categorizados, álbuns de fotos, agenda de eventos, banners etc.


Integração com redes sociais

O WPanel CMS vem com uma integração mínima com as redes sociais, dispondo de Likebox e comentários do Facebook, botões de compartilhamento do AddThis e configuração do código de rastreio do Google Analytics.

Também estamos usando um método para buscar os vídeos do Youtube pelo RSS do canal, facilmente configurado também no painel de controle e dispensando ter que cadastrar os vídeos manualmente no site.


Responsive Design

Todo o painel de controle foi desenvolvido usando Bootstrap e Jquery deixando o painel de controle responsivo, melhorando sua usabilidade nos dispositivos móveis do administrador.
